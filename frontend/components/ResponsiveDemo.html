🛠️ 响应式工具类 (Utility Classes)

  • .hidden-tablet - 在平板隐藏
  • .hidden-desktop - 在桌面端隐藏

🎨 CSS变量系统 (CSS Variables)

断点变量

--breakpoint-mobile: 768px; --breakpoint-tablet: 1024px;

容器宽度

--container-mobile: 100%; --container-tablet: 750px; --container-desktop: 1200px;

间距系统

--spacing-xs: 4px; --spacing-sm: 8px; --spacing-md: 16px; --spacing-lg: 24px; --spacing-xl: 32px; --spacing-xxl: 48px;

栅格间距

--grid-gutter-mobile: 8px; --grid-gutter-tablet: 16px; --grid-gutter-desktop: 24px;

💡 使用示例 (Usage Examples)

HTML 结构示例

<div class="container"> <div class="row"> <div class="col-xs-12 col-md-6 col-lg-4"> 内容区域 </div> </div> </div>

媒体查询示例

/* 移动端 */ @media (max-width: 767px) { } /* 平板 */ @media (min-width: 768px) and (max-width: 1023px) { } /* 桌面端 */ @media (min-width: 1024px) { }

CSS变量使用

.my-element { padding: var(--spacing-md); margin-bottom: var(--spacing-lg); font-size: var(--font-size-base); }
