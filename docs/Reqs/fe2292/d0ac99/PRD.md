# PRD — Design bill query API schema and database indexes

> 所属需求：账单查看与筛选

## 用户故事
As a 记账应用用户, I want to 通过多种条件组合查询历史账单记录, So that 我可以快速找到特定时间段、分类或金额范围内的账单，并导出数据用于分析或报税。

## 功能需求
- 时间筛选：支持快捷选项（今天/本周/本月）和自定义日期范围（起止日期）
- 分类筛选：支持单选或多选账单分类（餐饮/交通/购物等）
- 类型筛选：支持选择收入/支出/全部
- 金额范围筛选：支持输入最小值和最大值（支持留空表示不限）
- 关键词搜索：支持在备注字段中模糊匹配关键词
- 多条件组合：所有筛选条件可同时生效（AND 逻辑）
- 视图切换：支持列表视图（Table）和卡片视图（Card Grid）
- 数据导出：支持导出当前筛选结果为 CSV 或 Excel 文件
- 分页加载：列表视图支持分页，每页默认 20 条
- 排序功能：支持按时间/金额升序或降序排列

## 验收标准
- [ ] API 请求参数包含：start_date（ISO 8601 格式）、end_date、category_ids（数组）、type（income/expense/all）、min_amount、max_amount、keyword、page、page_size、sort_by、sort_order，所有参数均为可选
- [ ] API 响应时间 ≤ 500ms（数据量 < 10000 条时），≤ 1000ms（数据量 ≥ 10000 条时）
- [ ] 数据库索引覆盖：bills 表的 created_at、category_id、type、amount 字段建立复合索引，EXPLAIN 查询计划显示 Using index
- [ ] 时间范围筛选：选择「本月」时自动计算当月 1 日 00:00:00 至当前时间，返回结果的 created_at 字段均在此范围内
- [ ] 金额范围筛选：输入 min_amount=100, max_amount=500 时，返回结果的 amount 字段均满足 100 ≤ amount ≤ 500
- [ ] 关键词搜索：输入 keyword="午餐" 时，返回结果的 note 字段包含「午餐」（不区分大小写）
- [ ] 多条件组合：同时设置 type=expense + category_ids=[1,2] + keyword="打车" 时，返回结果同时满足三个条件（AND 逻辑）
- [ ] 空结果处理：筛选条件无匹配数据时，API 返回 {"data": [], "total": 0}，前端显示空状态图 + 文案「未找到符合条件的账单」
- [ ] 分页正确性：page=2, page_size=20 时返回第 21-40 条数据，响应包含 total（总条数）和 pages（总页数）字段
- [ ] 导出功能：点击「导出 CSV」后 2s 内生成文件并触发浏览器下载，文件名格式为 bills_YYYYMMDD_HHmmss.csv
- [ ] CSV 格式：包含表头（日期/分类/类型/金额/备注），金额保留 2 位小数，日期格式为 YYYY-MM-DD HH:mm:ss
- [ ] Excel 格式：包含表头，金额列格式为货币类型（¥符号 + 千分位逗号），日期列格式为 yyyy-mm-dd hh:mm
- [ ] 视图切换：点击视图切换按钮后 100ms 内完成切换，列表视图显示 Table 组件，卡片视图显示 Card Grid（每行 3 张卡片）
- [ ] 排序功能：点击「按金额降序」后，返回结果的 amount 字段按从大到小排列，相同金额按时间倒序
- [ ] 索引性能：执行包含 3 个筛选条件的查询时，数据库查询时间 ≤ 50ms（通过慢查询日志验证）
- [ ] 参数校验：传入非法日期格式（如 "2024-13-01"）时，API 返回 400 错误 + 错误信息「Invalid date format, expected ISO 8601」
- [ ] 并发支持：10 个用户同时发起查询请求时，所有请求均在 1s 内返回结果，无超时或 500 错误

## 边界条件（不做的事）
- 不包含：账单统计图表（由独立的统计模块负责）
- 不包含：账单编辑/删除功能（由账单管理模块负责）
- 暂不支持：导出 PDF 格式（Phase 2 考虑）
- 暂不支持：多用户协同筛选（仅支持单用户查询自己的账单）
- 暂不支持：保存筛选条件为预设（快捷筛选功能在 Phase 2 实现）
- 超出范围：账单数据的备份与恢复
- 超出范围：账单附件（图片/发票）的查询与预览

## 资产需求线索
- 图标：筛选图标（filter icon）、日历图标（calendar icon）、导出图标（download icon）、列表视图图标（list icon）、卡片视图图标（grid icon）、排序图标（sort icon）
- 插画：空状态插画（empty state illustration，尺寸 200x200px，风格简约）
- 暂无音效需求
- 暂无动画需求
