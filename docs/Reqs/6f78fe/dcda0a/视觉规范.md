# 视觉规范 — 用户信息更新 API 开发

> 所属需求：用户个人中心

## 整体风格
Modern flat design with subtle depth through shadows and borders. Primary indigo color (#4F46E5) for interactive elements and brand identity. Clean typography hierarchy using system fonts for optimal readability across devices. Generous whitespace (16-24px gaps) between form sections to reduce cognitive load. Smooth micro-interactions (150-300ms transitions) on hover/focus states to provide tactile feedback. Support both light and dark themes with WCAG AA contrast ratios (4.5:1 for text, 3:1 for UI components). Mobile-first responsive approach with touch-friendly targets (minimum 44x44px for buttons).

# Component Visual Specifications

## Form Input Field

### Text Input (Nickname)
- **Container**: Background `bg_primary`, border `1px solid border_default`, radius `md`, padding `sm md`
- **Typography**: Font size `base`, weight `normal`, color `text_primary`
- **Placeholder**: Color `text_secondary`, italic style
- **States**:
  - **Focus**: Border color `border_focus`, shadow `0 0 0 3px rgba(79, 70, 229, 0.1)`
  - **Error**: Border color `border_error`, helper text color `error`
  - **Disabled**: Background `bg_tertiary`, text color `text_disabled`, cursor not-allowed
- **Label**: Font size `sm`, weight `medium`, color `text_primary`, margin bottom `xs`
- **Helper Text**: Font size `xs`, color `text_secondary`, margin top `xs`
- **Character Counter**: Font size `xs`, color `text_secondary`, position absolute right

## Dark Mode Overrides

- **Backgrounds**: Use `dark_bg_primary`, `dark_bg_secondary`
- **Text**: Use `dark_text_primary`, `dark_text_secondary`
- **Borders**: Use `dark_border_default`
- **Shadows**: Reduce opacity by 50% or use `0 0 0 1px rgba(255,255,255,0.1)` for subtle borders
- **Primary Color**: Keep same for brand consistency
- **Input Focus**: Shadow color adjust to `rgba(79, 70, 229, 0.3)` for visibility
