# PRD — Design statistics data aggregation API schema

> 所属需求：统计报表与数据可视化

## 用户故事
作为用户，我需要 Design statistics data aggregation API schema

## 功能需求
- 设计统计数据聚合 API 的接口规范，包括：
- 收支趋势查询接口（支持 day/week/month 维度）
- 分类占比统计接口（支出/收入分别统计）
- 月度收支对比接口
- 年度统计总览接口
- 关键指标查询接口（本月总收入/总支出/结余）

定义请求参数、响应格式、错误码，输出 API 文档。

## 验收标准
- 功能可正常使用（待细化）

## 边界条件（不做的事）
- 暂无特殊限制
