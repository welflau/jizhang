# 视觉规范 — Create unified response format and exception handler middleware

> 所属需求：后端 API 服务搭建

## 整体风格
The visual design follows a technical documentation aesthetic with clear information hierarchy and status-driven color coding. Use monospace fonts for all JSON/code displays to maintain readability and professional appearance. Emphasize visual distinction between success (green), client errors (amber), and server errors (red) through consistent use of color-coded borders, badges, and status indicators. Maintain generous whitespace and clear component boundaries to support quick scanning of API response structures.

## 组件视觉规范

# Component Visual Specifications

## Response JSON Display Component

### Success Response Card
- **Background**: `bg_primary` (#FFFFFF)
- **Border**: 1px solid `border` (#E5E7EB), radius `md` (6px)
- **Padding**: `lg` (16px)
- **Status Indicator**: 8px circle, `success` color (#10B981), positioned top-left with `sm` margin
- **Code Block**:
  - Background: `code_bg` (#F3F4F6)
  - Text color: `code_text` (#1F2937)
  - Font: `font_family_code`, size `code` (13px)
  - Padding: `md` (12px)
  - Border radius: `sm` (4px)
- **Field Labels**: `text_secondary` (#6B7280), font size `sm` (12px), uppercase, letter-spacing 0.5px
- **Field Values**: `text_primary` (#111827), font size `base` (14px)

### Error Response Card
- **Background**: `bg_primary` (#FFFFFF)
- **Border**: 1px solid `error` (#EF4444), radius `md` (6px)
- **Box Shadow**: `shadow.error` (0 0 0 3px rgba(239, 68, 68, 0.1))
- **Padding**: `lg` (16px)
- **Status Indicator**: 8px circle, `error` color (#EF4444), positioned top-left with `sm` margin
- **Error Badge**: 
  - Background: rgba(239, 68, 68, 0.1)
  - Text color: `error` (#EF4444)
  - Padding: `xs` `sm` (4px 8px)
  - Border radius: `sm` (4px)
  - Font size: `sm` (12px), font-weight: 600
- **Code Block**: Same as success response

## Exception Type Badge
- **NotFoundError**: Background rgba(245, 158, 11, 0.1), text `warning` (#F59E0B)
- **ValidationError**: Background rgba(239, 68, 68, 0.1), text `error` (#EF4444)
- **SystemError**: Background rgba(107, 114, 128, 0.1), text `text_secondary` (#6B7280)
- **HTTPException**: Background rgba(59, 130, 246, 0.1), text `info` (#3B82F6)
- **Dimensions**: Height 24px, padding `xs` `md` (4px 12px), radius `md` (6px)
- **Typography**: Font size `sm` (12px), font-weight: 500

## Status Code Indicator
- **2xx Success**: Circle 20px, background `success` (#10B981), white checkmark icon 12px
- **4xx Client Error**: Circle 20px, background `warning` (#F59E0B), white alert icon 12px
- **5xx Server Error**: Circle 20px, background `error` (#EF4444), white cross icon 12px
- **Shadow**: `shadow.sm` (0 1px 2px 0 rgba(0, 0, 0, 0.05))

## API Documentation Panel
- **Background**: `bg_secondary` (#F9FAFB)
- **Border**: None
- **Padding**: `xl` (24px)
- **Section Headers**: 
  - Font size `heading` (18px)
  - Color `text_primary` (#111827)
  - Font-weight: 600
  - Margin-bottom: `lg` (16px)
- **Divider**: 1px solid `border` (#E5E7EB), margin `lg` (16px) vertical

## Interactive States

### Hover State (for copyable code blocks)
- **Background**: Darken `code_bg` by 5%
- **Cursor**: pointer
- **Transition**: background-color 150ms ease

### Focus State (for interactive elements)
- **Outline**: 2px solid `info` (#3B82F6)
- **Outline Offset**: 2px

### Loading State
- **Skeleton**: `bg_secondary` (#F9FAFB) with shimmer animation
- **Animation**: Linear gradient moving left-to-right, duration 1.5s
