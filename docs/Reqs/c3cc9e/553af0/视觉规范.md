# 视觉规范 — Create unified response format and exception handler middleware

> 所属需求：后端 API 服务搭建

## 整体风格
采用现代开发者友好的视觉风格，强调代码可读性和状态清晰度。使用语义化颜色系统（绿色=成功、红色=错误、黄色=警告、蓝色=信息）确保异常类型一目了然。所有代码展示区域使用等宽字体和浅灰背景，保持与主流 API 文档工具（Swagger/Postman）一致的视觉语言。响应格式展示采用卡片式布局，通过左侧色条和阴影效果强化状态识别，支持浅色/深色主题切换以适应不同开发环境。

## 组件视觉规范

# Visual Specification — Response Format & Exception Handler

## 1. Success Response Display
**用途**: API 成功响应的视觉呈现（开发者工具/文档示例）
- **背景色**: `bg_secondary` (#F9FAFB)
- **边框**: 1px solid `success` (#10B981)
- **内边距**: `md` (16px)
- **圆角**: `md` (6px)
- **字体**: `font_family_code`, `code` (13px)
- **文本颜色**: `code_text` (#1F2937)
- **状态标识**: 左侧 4px 宽绿色竖条 `success`

## 2. Error Response Display
**用途**: API 错误响应的视觉呈现
- **背景色**: `bg_primary` (#FFFFFF)
- **边框**: 1px solid `error` (#EF4444)
- **内边距**: `md` (16px)
- **圆角**: `md` (6px)
- **字体**: `font_family_code`, `code` (13px)
- **文本颜色**: `code_text` (#1F2937)
- **状态标识**: 左侧 4px 宽红色竖条 `error`
- **阴影**: `shadow.error` (错误状态高亮)

## 3. Exception Type Badge
**用途**: 异常类型标识（NotFoundError, ValidationError 等）
- **背景色**: 
  - NotFoundError: `warning` (#F59E0B) at 10% opacity
  - ValidationError: `error` (#EF4444) at 10% opacity
  - SystemError: `info` (#3B82F6) at 10% opacity
- **文本颜色**: 对应背景色的 100% 饱和度版本
- **内边距**: `xs` (4px) 水平, `xs` (4px) 垂直
- **圆角**: `sm` (4px)
- **字体**: `font_family_base`, `sm` (12px), font-weight: 500

## 4. Response Code Indicator
**用途**: HTTP 状态码视觉标识
- **2xx (Success)**: 
  - 文本颜色: `success` (#10B981)
  - 背景色: `success` at 10% opacity
- **4xx (Client Error)**: 
  - 文本颜色: `warning` (#F59E0B)
  - 背景色: `warning` at 10% opacity
- **5xx (Server Error)**: 
  - 文本颜色: `error` (#EF4444)
  - 背景色: `error` at 10% opacity
- **字体**: `font_family_code`, `base` (14px), font-weight: 600
- **内边距**: `sm` (8px) 水平, `xs` (4px) 垂直
- **圆角**: `sm` (4px)

## 5. JSON Response Container
**用途**: API 响应 JSON 数据展示容器
- **背景色**: `code_bg` (#F3F4F6)
- **边框**: 1px solid `border` (#E5E7EB)
- **内边距**: `md` (16px)
- **圆角**: `lg` (8px)
- **字体**: `font_family_code`, `code` (13px)
- **行高**: 1.6
- **最大高度**: 400px (超出滚动)
- **滚动条样式**: 8px 宽, `border` 颜色

## 6. Error Message Text
**用途**: 错误消息文本显示
- **字体**: `font_family_base`, `base` (14px)
- **文本颜色**: `text_primary` (#111827)
- **行高**: 1.5
- **错误关键词高亮**: `error` (#EF4444), font-weight: 500
