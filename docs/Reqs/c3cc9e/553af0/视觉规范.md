# 视觉规范 — Create unified response format and exception handler middleware

> 所属需求：后端 API 服务搭建

## 整体风格
采用现代开发者工具风格，以清晰的状态色彩系统（绿色成功、红色错误、黄色警告、蓝色信息）为核心，确保 API 响应信息的可读性。代码块使用等宽字体和语法高亮，保持专业的技术文档美感。整体设计简洁克制，避免过度装饰，聚焦于信息的快速识别和错误的准确定位。
