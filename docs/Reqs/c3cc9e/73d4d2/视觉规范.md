# 视觉规范 — Implement JWT authentication middleware

> 所属需求：后端 API 服务搭建

## 整体风格
The visual design for JWT authentication middleware components follows a **modern, developer-focused aesthetic** with high contrast for readability in both light and dark themes. Use **monospace fonts for token display** to enhance technical clarity, and apply **color-coded status indicators** (green for authenticated, red for errors, amber for warnings) to provide instant visual feedback. All interactive elements should have **subtle shadows and smooth transitions** (150-200ms) to create a polished, professional feel suitable for API documentation and admin dashboards.

# Component Visual Specification — JWT Authentication Middleware

## 1. Authentication Status Badge
**Purpose**: Display token validity status in API documentation or admin dashboard

- **Authenticated State**:
  - Background: `success` (#10B981) with 10% opacity
  - Border: 1px solid `success`
  - Text: `success`, font-size `size_sm`, weight `weight_medium`
  - Icon: checkmark circle, 16px, `success` color
  - Padding: `xs` `sm`
  - Border-radius: `radius.full`

- **Unauthenticated State**:
  - Background: `error` (#EF4444) with 10% opacity
  - Border: 1px solid `error`
  - Text: `error`, font-size `size_sm`, weight `weight_medium`
  - Icon: x-circle, 16px, `error` color
  - Padding: `xs` `sm`
  - Border-radius: `radius.full`

- **Token Expired State**:
  - Background: `warning` (#F59E0B) with 10% opacity
  - Border: 1px solid `warning`
  - Text: `warning`, font-size `size_sm`, weight `weight_medium`
  - Icon: clock, 16px, `warning` color
  - Padding: `xs` `sm`
  - Border-radius: `radius.full`

## 2. Token Display Component
**Purpose**: Show JWT token in developer tools or testing interface

- **Container**:
  - Background: `surface_light` / `surface_dark`
  - Border: 1px solid `border_light` / `border_dark`
  - Border-radius: `radius.md`
  - Padding: `md`
  - Shadow: `shadow.sm`

- **Token Text**:
  - Font-family: `font_family_mono`
  - Font-size: `size_sm`
  - Color: `text_primary_light` / `text_primary_dark`
  - Word-break: break-all
  - Line-height: `line_height_relaxed`
  - Max-height: 120px with vertical scroll

- **Copy Button**:
  - Background: transparent, hover `primary` with 10% opacity
  - Border: 1px solid `border_light` / `border_dark`
  - Icon: clipboard, 16px, `text_secondary_light` / `text_secondary_dark`
  - Padding: `xs`
  - Border-radius: `radius.sm`
  - Transition: `transition.fast`

## 3. Error Message Component
**Purpose**: Display authentication errors (401 Unauthorized)

- **Container**:
  - Background: `error` with 5% opacity
  - Border-left: 4px solid `error`
  - Padding: `md`
  - Border-radius: `radius.md`
  - Margin-bottom: `md`

- **Title**:
  - Font-size: `size_base`
  - Font-weight: `weight_semibold`
  - Color: `error`
  - Margin-bottom: `xs`

- **Message**:
  - Font-size: `size_sm`
  - Color: `text_primary_light` / `text_primary_dark`
  - Line-height: `line_height_normal`

- **Icon**:
  - alert-circle, 20px, `error` color
  - Position: top-left, aligned with title

## 4. Token Expiration Timer
**Purpose**: Show remaining time before token expires

- **Container**:
  - Display: inline-flex, align-items center
  - Gap: `xs`

- **Icon**:
  - clock, 14px, `text_secondary_light` / `text_secondary_dark`

- **Time Text**:
  - Font-size: `size_sm`
  - Color: `text_secondary_light` / `text_secondary_dark`
  - Font-variant-numeric: tabular-nums

- **Warning State** (< 5 minutes remaining):
  - Icon color: `warning`
  - Text color: `warning`
  - Font-weight: `weight_medium`

- **Critical State** (< 1 minute remaining):
  - Icon color: `error`
  - Text color: `error`
  - Font-weight: `weight_semibold`
  - Animation: pulse every 1s

## 5. Role Badge
**Purpose**: Display user role from JWT payload

- **Admin Role**:
  - Background: `primary` with 15% opacity
  - Text: `primary`, font-size `size_xs`, weight `weight_semibold`, uppercase
  - Padding: `xs` `sm`
  - Border-radius: `radius.sm`

- **User Role**:
  - Background: `text_secondary_light` / `text_secondary_dark` with 10% opacity
  - Text: `text_secondary_light` / `text_secondary_dark`, font-size `size_xs`, weight `weight_medium`, uppercase
  - Padding: `xs` `sm`
  - Border-radius: `radius.sm`

- **Guest Role**:
  - Background: `border_light` / `border_dark`
  - Text: `text_secondary_light` / `text_secondary_dark`, font-size `size_xs`, weight `weight_normal`, uppercase
  - Padding: `xs` `sm`
  - Border-radius: `radius.sm`

## 6. Protected Route Indicator
**Purpose**: Show lock icon for protected endpoints in API documentation

- **Icon**:
  - lock-closed, 16px, `primary` color
  - Display: inline-block
  - Vertical-align: middle

- **Tooltip** (on hover):
  - Background: `surface_dark` (force dark even in light theme)
  - Text: white, font-size `size_xs`
  - Padding: `xs` `sm`
  - Border-radius: `radius.sm`
  - Shadow: `shadow.lg`
  - Content: "Requires authentication"

## 7. Refresh Token Button
**Purpose**: Manual token refresh action

- **Default State**:
  - Background: `primary`
  - Text: white, font-size `size_sm`, weight `weight_medium`
  - Icon: refresh-cw, 16px, white
  - Padding: `sm` `md`
  - Border-radius: `radius.md`
  - Shadow: `shadow.sm`
  - Transition: `transition.base`

- **Hover State**:
  - Background: `primary_hover`
  - Shadow: `shadow.md`
  - Transform: translateY(-1px)

- **Loading State**:
  - Background: `primary` with 70% opacity
  - Icon: spinning animation, 360deg in 1s linear infinite
  - Cursor: not-allowed

- **Disabled State**:
  - Background: `border_light` / `border_dark`
  - Text: `text_secondary_light` / `text_secondary_dark`
  - Cursor: not-allowed
  - Shadow: none
