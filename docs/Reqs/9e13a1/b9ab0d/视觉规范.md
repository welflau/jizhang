# 视觉规范 — Create budgets table schema and migration

> 所属需求：预算管理功能

## 整体风格
Modern, data-centric design with emphasis on clarity and technical precision. Use monospaced fonts for code, SQL, and numeric data to enhance readability. Employ a clean, minimal aesthetic with subtle borders and shadows to create visual hierarchy without clutter. Color palette centers on indigo primary (#4F46E5) with semantic colors for success/error states. Support both light and dark themes with high contrast ratios (WCAG AA minimum). Prioritize information density on desktop while maintaining touch-friendly spacing (min 44px tap targets) on mobile.

## Responsive Breakpoints

- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px

### Mobile Adjustments
- Table: horizontal scroll enabled
- Padding: reduce to `spacing.sm`
- Font size: reduce by 1 step (base → sm)
- Toast: full width with `spacing.sm` margin
