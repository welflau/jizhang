# 视觉规范 — Create budgets table schema and migration

> 所属需求：预算管理功能

## 整体风格
Modern data-centric design with developer-friendly aesthetics. Use monospace fonts for technical identifiers (table names, field names, SQL keywords) to enhance code readability. Employ a clean, card-based layout with subtle shadows and clear visual hierarchy. Color-coded badges and tags help users quickly identify field types, constraints, and index information. The interface should feel professional and trustworthy, suitable for backend/database management tools, with a focus on information density without overwhelming the user. Support both light and dark themes with high contrast ratios for accessibility.

## Responsive Breakpoints

### Mobile (< 640px)
- Container padding: `md` (16px)
- Font sizes: reduce by 1 step (base → sm, lg → base)
- Button full width
- Toast width: calc(100vw - 32px)

### Tablet (640px - 1024px)
- Container max width: 720px
- Maintain desktop spacing

### Desktop (> 1024px)
- Container max width: 800px
- Full spacing and typography scale
