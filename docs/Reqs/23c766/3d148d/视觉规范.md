# 视觉规范 — Frontend: Add data backup and restore UI

> 所属需求：数据备份与恢复

## 整体风格
采用 Ant Design 设计语言，遵循现代企业级应用的视觉规范。整体风格简洁专业，以蓝色 (#1890ff) 为主色调，强调功能性和可用性。危险操作（清空数据）使用红色 (#ff4d4f) 高亮警示，确保用户能清晰识别高风险操作。所有交互状态（hover/active/loading/disabled）均有明确视觉反馈，响应式布局适配桌面端和移动端，保持一致的视觉体验。

# Component Visual Specification

## Data Management Section Container
- **Background**: `bg_container` (#fafafa)
- **Border**: 1px solid `border_light` (#f0f0f0)
- **Border Radius**: `lg` (8px)
- **Padding**: `lg` (24px)
- **Margin Top**: `xl` (32px) — 与上方访问统计区域分离
- **Box Shadow**: `card`

## Section Title
- **Text**: "数据管理" or "Data Management"
- **Font Size**: `font_size_heading` (20px)
- **Font Weight**: `font_weight_semibold` (600)
- **Color**: `text_primary` (#000000d9)
- **Margin Bottom**: `md` (16px)

## Export Button
- **Type**: Default button
- **Icon**: `DownloadOutlined` (24px, positioned left)
- **Text**: "导出数据" / "Export Data"
- **Background**: `bg_base` (#ffffff)
- **Border**: 1px solid `border_base` (#d9d9d9)
- **Border Radius**: `base` (4px)
- **Padding**: 4px 15px
- **Height**: 32px
- **Font Size**: `font_size_base` (14px)
- **Color**: `text_primary`
- **Hover State**: Border color → `primary` (#1890ff), Color → `primary`
- **Active State**: Border color → `primary_active` (#096dd9), Color → `primary_active`
- **Loading State**: Icon replaced with Spin, button disabled, cursor not-allowed

## Import Section
- **Layout**: Vertical flex container
- **Spacing**: `md` (16px) between elements
- **Margin Top**: `lg` (24px)

### Upload Area
- **Component**: Ant Design Upload.Dragger
- **Background**: `bg_base` (#ffffff)
- **Border**: 1px dashed `border_base` (#d9d9d9)
- **Border Radius**: `base` (4px)
- **Padding**: `lg` (24px)
- **Min Height**: 120px
- **Hover State**: Border color → `primary` (#1890ff)
- **Icon**: `UploadOutlined` (48px, color `primary`)
- **Primary Text**: "点击或拖拽 JSON 文件到此区域" / "Click or drag JSON file to this area"
  - Font Size: `font_size_base` (14px)
  - Color: `text_primary`
- **Secondary Text**: "仅支持 .json 格式文件" / "Only .json format supported"
  - Font Size: `font_size_sm` (12px)
  - Color: `text_secondary` (#00000073)

### Import Result Message
- **Type**: Ant Design message component
- **Success**: Green checkmark icon + "导入成功：X 条记录" / "Import successful: X records"
  - Background: `success` tint
  - Duration: 3s
- **Error**: Red cross icon + "导入失败：Y 条记录失败" / "Import failed: Y records failed"
  - Background: `danger` tint
  - Duration: 3s

## Loading State (Global)
- **Spin Icon**: Ant Design default spinner
- **Color**: `primary` (#1890ff)
- **Size**: 16px (in buttons), 24px (standalone)
- **Animation**: Rotate 360deg, duration `duration_base` (0.2s), infinite

## Responsive Breakpoints
- **Desktop (≥992px)**: Buttons in horizontal row, gap `md` (16px)
- **Tablet (768px-991px)**: Buttons in horizontal row, gap `sm` (8px)
- **Mobile (<768px)**: Buttons stacked vertically, full width, gap `sm` (8px)

## Accessibility
- **Focus State**: 2px solid `primary` outline, offset 2px
- **ARIA Labels**: All buttons have aria-label attributes
- **Keyboard Navigation**: Tab order: Export → Upload → Clear
- **Screen Reader**: Upload area has aria-describedby for format hint
