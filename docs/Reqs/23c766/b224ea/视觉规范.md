# 视觉规范 — E2E: Test data backup and restore workflow

> 所属需求：数据备份与恢复

## 整体风格
Modern, clean interface with emphasis on data safety and clarity. Use indigo primary color (#4F46E5) to convey trust and professionalism. Maintain generous whitespace and clear visual hierarchy to reduce cognitive load during critical backup/restore operations. All interactive elements should have clear hover/active states with smooth transitions (200ms) to provide immediate feedback. Error states use red (#EF4444) with clear iconography; success states use green (#10B981). Typography is system-native for optimal readability across platforms.

# Component Visual Specifications

## Button (Primary Action)
- **Background**: `colors.primary` (#4F46E5)
- **Text**: White (#FFFFFF), `typography.md` (14px), `font_weight_medium` (500)
- **Padding**: `spacing.sm` (8px) vertical, `spacing.md` (16px) horizontal
- **Border Radius**: `radius.md` (8px)
- **Shadow**: `shadow.sm`
- **States**:
  - Hover: Background `colors.primary_hover` (#4338CA), shadow `shadow.md`
  - Active: Background `colors.primary_active` (#3730A3)
  - Loading: Background `colors.primary`, opacity 0.7, spinner icon white 16px
  - Disabled: Background `colors.text_disabled` (#9CA3AF), cursor not-allowed
- **Transition**: All properties `transition.base` (200ms) with `easing`

## Button (Secondary Action)
- **Background**: `colors.bg_primary` (#FFFFFF)
- **Text**: `colors.text_primary` (#111827), `typography.md` (14px), `font_weight_medium` (500)
- **Border**: 1px solid `colors.border` (#E5E7EB)
- **Padding**: `spacing.sm` (8px) vertical, `spacing.md` (16px) horizontal
- **Border Radius**: `radius.md` (8px)
- **States**:
  - Hover: Background `colors.bg_secondary` (#F9FAFB), border `colors.primary`
  - Active: Background `colors.bg_tertiary` (#F3F4F6)
  - Disabled: Text `colors.text_disabled`, cursor not-allowed

## Counter Display
- **Container**: Background `colors.bg_secondary` (#F9FAFB), padding `spacing.lg` (24px), border-radius `radius.lg` (12px)

- **Value**: `colors.text_primary` (#111827), `typography.heading_lg` (24px), `font_weight_bold` (700), margin-top `spacing.xs` (4px)
