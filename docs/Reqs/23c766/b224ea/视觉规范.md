# 视觉规范 — E2E: Test data backup and restore workflow

> 所属需求：数据备份与恢复

## 整体风格
Modern, clean interface with emphasis on data safety and clarity. Use ample whitespace and clear visual hierarchy to guide users through the backup/restore workflow. Primary actions (export, import) use bold primary color (#4F46E5) while destructive actions (clear data) use subtle secondary styling with confirmation dialogs. Toast notifications provide immediate feedback with color-coded borders (green for success, red for errors). Maintain consistent 8px spacing grid and smooth transitions (150-200ms) for all interactive elements.
