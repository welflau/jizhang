# 视觉规范 — E2E: Test data backup and restore workflow

> 所属需求：数据备份与恢复

## 整体风格
Modern, clean, and functional design following contemporary web app standards. Use a restrained color palette with indigo primary (#4F46E5) for actions, green for success states, and red for destructive/error states. Prioritize clarity and readability with ample whitespace (16-24px gaps between major sections). All interactive elements should have clear hover/focus states with smooth transitions (200ms). Maintain consistent 8px grid alignment for spacing. Toast notifications slide in from top-center with subtle shadow for depth. Confirmation dialogs use centered modal overlay with backdrop blur effect. Button hierarchy is clear: primary actions use filled buttons, secondary actions use outlined buttons. Icons are 24px outline style from Heroicons for consistency. Typography uses system font stack for optimal performance and native feel across platforms.

## 组件视觉规范

# Component Visual Specifications

## Button (Primary Action)
- **Background**: `colors.primary` (normal), `colors.primary_hover` (hover), `colors.primary_active` (active)
- **Text**: `#FFFFFF`, `typography.size_base`, `typography.weight_medium`
- **Padding**: `spacing.sm` (vertical), `spacing.md` (horizontal)
- **Border**: None
- **Border Radius**: `radius.md`
- **Shadow**: `shadow.sm` (normal), `shadow.md` (hover)
- **Transition**: `transition.base`
- **States**:
  - Loading: Background unchanged, add spinner icon, cursor `wait`
  - Disabled: Background `#E5E7EB`, text `colors.text_disabled`, cursor `not-allowed`

## Button (Secondary Action)
- **Background**: `colors.bg_primary` (normal), `colors.bg_secondary` (hover)
- **Text**: `colors.text_primary`, `typography.size_base`, `typography.weight_medium`
- **Padding**: `spacing.sm` (vertical), `spacing.md` (horizontal)
- **Border**: `1px solid colors.border_default`
- **Border Radius**: `radius.md`
- **Transition**: `transition.base`

## Toast Notification
- **Background**: 
  - Success: `colors.success`
  - Error: `colors.error`
  - Warning: `colors.warning`
- **Text**: `#FFFFFF`, `typography.size_sm`, `typography.weight_normal`
- **Padding**: `spacing.md`
- **Border Radius**: `radius.lg`
- **Shadow**: `shadow.toast`
- **Min Width**: `280px`
- **Max Width**: `400px`
- **Icon**: 20px, aligned left with `spacing.sm` gap
- **Animation**: Slide in from top, duration `transition.slow`

## Confirmation Dialog
- **Overlay**: `colors.overlay`
- **Container Background**: `colors.bg_primary`
- **Container Padding**: `spacing.xl`
- **Container Border Radius**: `radius.lg`
- **Container Shadow**: `shadow.lg`
- **Max Width**: `480px`
- **Title**: `colors.text_primary`, `typography.size_xl`, `typography.weight_semibold`
- **Body Text**: `colors.text_secondary`, `typography.size_base`, `typography.line_height_relaxed`
- **Button Group**: Flex row, gap `spacing.sm`, justify end
- **Animation**: Fade in overlay + scale in dialog, duration `transition.base`

## Counter Display
- **Background**: `colors.bg_secondary`
- **Text**: `colors.text_primary`, `typography.size_2xl`, `typography.weight_bold`
- **Padding**: `spacing.lg`
- **Border**: `1px solid colors.border_default`
- **Border Radius**: `radius.md`
- **Label**: `colors.text_secondary`, `typography.size_sm`, `typography.weight_normal`
- **Spacing**: Label below counter with `spacing.xs` gap

## File Upload Area
- **Background**: `colors.bg_tertiary` (normal), `colors.bg_secondary` (hover/drag-over)
- **Border**: `2px dashed colors.border_default` (normal), `2px dashed colors.primary` (drag-over)
- **Border Radius**: `radius.lg`
- **Padding**: `spacing.xl`
- **Text**: `colors.text_secondary`, `typography.size_base`
- **Icon**: 48px, `colors.text_disabled`
- **Transition**: `transition.fast`
- **States**:
  - Drag Over: Border color `colors.primary`, background `colors.bg_secondary`
  - Error: Border color `colors.error`, show error message in `colors.error`

## Data List/Table
- **Background**: `colors.bg_primary`
- **Border**: `1px solid colors.border_default`
- **Border Radius**: `radius.md`
- **Row Padding**: `spacing.md`
- **Row Hover**: Background `colors.bg_secondary`
- **Header**: Background `colors.bg_tertiary`, text `colors.text_primary`, `typography.weight_semibold`
- **Cell Text**: `colors.text_primary`, `typography.size_sm`
