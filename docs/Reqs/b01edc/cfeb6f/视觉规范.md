## 整体风格
The visual design follows Ant Design 5.x design language with a clean, professional aesthetic suitable for enterprise web applications. The color palette centers on Ant Design's signature blue (#1890ff) as the primary action color, paired with neutral grays for text hierarchy and layout backgrounds. Typography uses system font stacks for optimal cross-platform rendering, with clear size and weight distinctions between headings and body text. Spacing follows an 8px grid system (4/8/16/24/32/48) for consistent rhythm. Interactive elements feature subtle shadows and smooth transitions (0.2s cubic-bezier easing) to provide tactile feedback. The overall style is modern, minimalist, and accessibility-focused, ensuring readability and usability across devices.
