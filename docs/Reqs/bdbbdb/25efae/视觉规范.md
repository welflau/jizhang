# 视觉规范 — Design category icon set and color palette

> 所属需求：分类管理系统

## 整体风格
**Visual Style**: Modern minimalist line icon system with 1.5px consistent stroke weight, 2px corner radius for rounded elements, and geometric simplicity. Icons follow a 24x24px grid system with optical alignment for visual balance. Color palette uses vibrant yet accessible hues (WCAG AA compliant) with semantic categorization—warm greens for income, varied saturated colors for expense types, and neutral grays for system elements. Overall aesthetic is clean, friendly, and suitable for financial applications requiring quick visual scanning and cross-cultural recognition. Design language prioritizes clarity over decoration, ensuring icons remain legible at small sizes (16px) and scalable to larger contexts (32px+) without loss of detail.

## 组件视觉规范

# Component Visual Specification

## CategoryIcon Component

### Base Specifications
- **Dimensions**: 24x24px viewBox (default), scalable to 16px/32px
- **Stroke Width**: 1.5px (consistent across all icons)
- **Corner Radius**: 2px for rounded elements
- **Color**: Single-color fill, passed via props (default: currentColor)
- **File Format**: SVG, optimized < 2KB per icon

### Visual States
1. **Default**: Icon color from props, opacity 1.0
2. **Hover**: Scale 1.05, add subtle shadow (0 2px 8px rgba(0,0,0,0.1))
3. **Active/Selected**: Background circle with 10% opacity of icon color, icon remains solid
4. **Disabled**: Opacity 0.4, cursor not-allowed

### Icon Container (when used with background)
- **Shape**: Circle or rounded square (8px radius)
- **Size**: 40x40px (icon 24px centered)
- **Background**: Color at 10% opacity or solid color
- **Padding**: 8px around icon
- **Border**: None (background provides visual boundary)

### Typography Pairing
- **Label Font**: 12px, font-weight 500, color gray-600
- **Label Position**: Below icon, centered, 4px gap
- **Category Name**: 14px, font-weight 600, color gray-900

## Color Palette Display

### Color Swatch Component
- **Swatch Size**: 48x48px rounded square (8px radius)
- **Border**: 1px solid gray-200
- **Label Layout**: 
  - Color name: 12px semibold, gray-900
  - Hex value: 11px regular, gray-600
  - Contrast ratio badge: 10px, white text on gray-700 background
- **Hover State**: Lift effect with shadow (0 4px 12px rgba(0,0,0,0.15))

### Accessibility Indicators
- **AA Badge**: Green checkmark icon + "AA" text for colors meeting WCAG 2.1 AA (contrast ≥ 4.5:1)
- **Warning Icon**: Amber alert icon for colors requiring text labels (contrast 3:1-4.5:1)
- **Color-blind Safe**: Blue info icon for deuteranopia/protanopia friendly combinations

---

## Icon Grid Layout

### Grid Specifications
- **Columns**: 4 columns on desktop (≥1024px), 3 on tablet (768-1023px), 2 on mobile (<768px)
- **Gap**: 12px between items
- **Item Structure**: Icon (40x40px container) + Label below
- **Alignment**: Center-aligned within grid cell

### Section Headers
- **Text**: 16px semibold, gray-900, 24px margin-bottom
- **Divider**: 1px solid gray-200, full width

## Responsive Behavior

### Mobile (<768px)
- Icon size: 24px (no scaling to 32px)
- Grid: 2 columns
- Tables: Horizontal scroll or card layout
- Touch target: Minimum 44x44px for interactive icons

### Tablet (768-1023px)
- Icon size: 24px default, 32px for featured categories
- Grid: 3 columns
- Tables: Full width with adjusted column proportions

### Desktop (≥1024px)
- Icon size: 24px default, 32px option available
- Grid: 4 columns
- Tables: Full specifications visible
- Hover effects enabled
