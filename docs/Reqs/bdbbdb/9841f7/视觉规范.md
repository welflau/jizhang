# 视觉规范 — Create categories database table and migration

> 所属需求：分类管理系统

## 整体风格
Modern flat design with clean lines and subtle depth through shadows. Embrace a dual-theme approach (light/dark) with high contrast for accessibility. Use indigo (#4F46E5) as primary brand color with semantic colors for income (green #10B981) and expense (red #EF4444). Typography is system-native sans-serif for optimal readability. Micro-interactions with smooth 200ms transitions enhance user feedback without distraction. Visual hierarchy is established through consistent spacing scale (4px base unit) and weight variations rather than color alone.

# Component Visual Specifications

## Database Schema Visualization Component

### Table Card
- **Background**: Light mode `#FFFFFF`, Dark mode `#374151`
- **Border**: 1px solid, Light mode `#E5E7EB`, Dark mode `#4B5563`
- **Border Radius**: `8px` (md)
- **Padding**: `16px` (md)
- **Shadow**: `shadow-md` on hover
- **Typography**: Table name uses `heading-md` (20px), weight `600`

### Field Row
- **Layout**: Flex row, space-between alignment
- **Padding**: `8px` (sm) vertical, `12px` horizontal
- **Border Bottom**: 1px solid, Light mode `#F3F4F6`, Dark mode `#4B5563`
- **Field Name**: `base` (14px), weight `500`, color `text_primary`
- **Field Type**: `sm` (12px), weight `400`, color `text_secondary`
- **Hover State**: Background `#F9FAFB` (light) / `#4B5563` (dark)

### Constraint Badge
- **Background**: 
  - Primary Key: `#EEF2FF` (light) / `#3730A3` (dark)
  - Foreign Key: `#FEF3C7` (light) / `#92400E` (dark)
  - Unique: `#DBEAFE` (light) / `#1E3A8A` (dark)
  - NOT NULL: `#FEE2E2` (light) / `#991B1B` (dark)
- **Text Color**: Corresponding darker/lighter shade
- **Padding**: `2px 8px`
- **Border Radius**: `4px` (sm)
- **Font Size**: `11px` (xs)
- **Font Weight**: `500`

### Index Indicator
- **Icon Size**: 16px
- **Color**: `#F59E0B` (warning)
- **Tooltip Background**: `#1F2937` (dark) / `#FFFFFF` (light)

### Type Enum Badge
- **Income**: Background `#D1FAE5`, Text `#065F46`, Border `#10B981`

- **Padding**: `4px 12px`
- **Border**: 1px solid
- **Border Radius**: `full` (9999px)
- **Font Size**: `12px` (sm)
- **Font Weight**: `500`

- **Success**: Border `#10B981`, Icon `#10B981`

### Animation
